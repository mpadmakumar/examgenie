# 🎓 ExamGenie — Tamil AI Tutor

> Free AI-powered tutor for Tamil medium students
> தமிழ் மாணவர்களுக்கான இலவச AI tutor

![License](https://img.shields.io/badge/license-MIT-green)
![Free](https://img.shields.io/badge/cost-FREE-brightgreen)
![Groq](https://img.shields.io/badge/powered%20by-Groq-orange)

## ✨ Features
- 🗣️ Tamil + English bilingual answers
- 🏆 Quiz Mode — MCQ with levels (Easy/Mid/Pro)
- 🎯 Adaptive Learning — AI tracks weak topics
- 💬 Multiple chat history
- 📚 6 subjects — Maths, Physics, Chemistry, Biology, History, Geography
- 🆓 100% Free — Groq API + Flask + React

## 🛠️ Tech Stack
- **Frontend**: React JS
- **Backend**: Flask (Python)  
- **AI**: Groq API (Llama 3.3 70B)
- **Cost**: ₹0

## 🚀 Quick Start

### Backend
\`\`\`bash
cd examgenie
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py
\`\`\`

### Frontend
\`\`\`bash
cd frontend
npm install
npm start
\`\`\`
